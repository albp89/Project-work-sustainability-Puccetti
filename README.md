# Collemattoni – Bilancio di Sostenibilità

Mini sito statico realizzato per presentare in modo sintetico
le principali informazioni del Bilancio di Sostenibilità 2024
dell’azienda Collemattoni. Il sito permette inoltre di scaricare il Report in formato PDF

Il progetto è sviluppato in HTML e CSS.

## Struttura
- `index.html` – pagina principale
- `styles.css` – foglio di stile
- asset grafici (logo e immagini)

## Utilizzo
Aprire il file `index.html` in un browser oppure cliccare direttamente sul link
