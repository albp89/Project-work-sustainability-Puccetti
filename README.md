# Sito per il download del Bilancio di Sostenibilità dell'azienda Collemattoni

Mini sito statico realizzato per presentare in modo sintetico
i contenuti principali del Bilancio di Sostenibilità 2024
dell’azienda Collemattoni e consentirne il download in formato PDF.

Il progetto è sviluppato esclusivamente in HTML e CSS.

## Obiettivo del progetto
L’obiettivo del sito è comunicare in modo chiaro e accessibile
le principali iniziative e i dati chiave relativi al percorso di sostenibilità dell’azienda,
attraverso una struttura semplice e una grafica coerente con l’identità visiva del brand.

## Struttura del progetto
- `index.html` – pagina principale del sito
- `styles.css` – foglio di stile
- asset grafici – logo azienda
- `Bilancio_2024.pdf` – report di sostenibilità scaricabile

## Utilizzo
Il sito può essere visualizzato:
- aprendo il file `index.html` in un browser
- oppure tramite GitHub Pages al seguente indirizzo:  
  https://albp89.github.io/Project-work-sustainability-Puccetti/
